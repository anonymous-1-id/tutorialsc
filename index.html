📘 BAB 1: Apa Itu Bot WhatsApp?

Bot WhatsApp adalah program otomatis yang dapat membaca dan membalas pesan secara real-time. Bot ini digunakan untuk berbagai tujuan seperti auto-reply, sistem layanan pelanggan, promosi, hingga sistem panel. Kita akan membuat bot dengan sistem handler case dan konfigurasi global berbasis file setting.js.

📁 BAB 2: Persiapan Awal

  • Install Node.js
  • Gunakan Termux (Android) atau CMD / Terminal
  • Install npm i baileys atau library WA lainnya
  • Edit pakai VSCode atau editor kesukaanmu

🗂️ BAB 3: Struktur File & Folder

📁 mybot-wa/
├── index.js             # File utama untuk menjalankan bot
├── bot.js               # Semua case switch (menu, ytmp3, dll) berada di sini
├── setting.js           # Konfigurasi global seperti nomor owner, nama bot
├── package.json         # Konfigurasi project npm
├── database/            # Folder untuk menyimpan media (audio, gambar, dsb.)
│   ├── image/

Semua fitur bot ditulis langsung di file bot.js dengan sistem func-case (switch). Konfigurasi seperti owner dan nama bot diatur di setting.js.
